Astrophotography

What is it?

Combining my passion for photography with the awe of space, I have been learning how to best capture starscapes, to showcase the beauty of the galaxies around us.

Why?

Outside of school, I've always had a passion for photography, and recently wanted to explore Astrophotography. When I would go on camping trips with my family, we would generally go to very remote areas, with low levels of light pollution. This meant the stars at night were very vivid, and I wanted to learn how to capture them.

I had very little experience with python programs, and had only used Raspberry pi as a personal home media system, so the entire project was incredibly daunting. This necessitated me to research and understand what would be the best possible solution for the problem, and initially, I created the system using an open source project named MotionEyeOS. When I implemented this software however, I was met with multi-second latency and very low resolution images, forcing me to dive deeper in the world of python programming. I eventually found a protocal called mJPEG, which allowed for streaming to a website via python's http.server module. From there it took some more research to create scripts to run the programs on boot, and to get the correct network configuration setup. The next steps are to streamline the OS, and we plan on developing a dedicated chipset and creating a full fledged product.

What I've learned

What I used

  • Raspberry Picamera libraries
  • IP address and network configuration knowledge
  • Python programming to create streaming server
  • MotionEyeOS
  • Linux scripting to run programs on boot
